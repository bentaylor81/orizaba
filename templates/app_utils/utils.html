Utilities

Product Details Page > Quantity in Stock > Orizaba Stock Qty

  • Sets the stock quantity on Orizaba to the quantity in Unleashed.
  • Used to reset the stock figure to Unleashed if any errors in stock calculation has been fixed.

Product Details Page > Stock Movement Table > Stock Qty to Null

  • Sets the running Stock Quantity column in the Stock Movement table in the products page all to null.
  • This is when you need to reset the Orizaba stock qty above and start again with no running total.

Unleashed Page > Date Added

  • Set the date added to the order date in the stock movement table.

Set Firstname / Lastnamse

  • Split the delivery and billing name and set to respective firstname / lastname fields.

Refresh ShipTheory Bearer Token

  • Generate a new bearer token for the ShipTheory API.
  • Add the bearer token to the config vars in Heroku.
